# 🗺️ Cursus - User Story Mapping

## 📌 Visión del Producto
Plataforma integral para estudiantes universitarios, enfocada en la gestión académica, métodos de estudio eficientes y seguimiento administrativo, centralizando las herramientas necesarias para la carrera.

---

## 🏗️ Épicas y Flujos de Usuario (User Journeys)

### 1️⃣ Autenticación y Perfil
*Flujos:* Registro/Login, Recuperación de contraseña, Personalización del entorno.

### 2️⃣ Gestión Académica
*Flujos:* Visualización del plan de estudios, Carga de materias y notas, Calendario y Horarios, Visualización de Progreso.

### 3️⃣ Área de Estudio y Productividad
*Flujos:* Pomodoro interactivo, Gestión Kanban de tareas, Marcadores y recursos.

### 4️⃣ Aprendizaje Activo
*Flujos:* Creación de mazos (Decks), Generación de Flashcards (Manual/IA), Sesiones de repaso y métricas.

### 5️⃣ Finanzas, Comunidad y Administración
*Flujos:* Estado de cuotas, Portal de Beneficios, Formulario de Contacto, Panel de Administración.

## 🚀 Mapa de Entregas (Releases)

### 🟢 MVP / Release 1.0 (Core Functionality)
*(Base funcional ya desarrollada o en estado básico de implementación)*

| Épica | Historias de Usuario (User Stories) |
| :--- | :--- |
| **Autenticación y Perfil** | <ul><li>Como estudiante, quiero iniciar sesión y registrarme para acceder a mi área personal.</li><li>Como estudiante, quiero personalizar mi avatar y fondo (Custom BG) para sentir propio mi espacio.</li></ul> |
| **Gestión Académica** | <ul><li>Como estudiante, quiero ver mis materias y cargar mis notas para seguir mi historial.</li><li>Como estudiante, quiero visualizar mis horarios de cursada básicos.</li><li>Como estudiante, quiero gestionar vencimientos y alertas (académicas, administrativas y personales).</li><li>Como estudiante, quiero tener un panel básico de progreso de mi carrera.</li></ul> |
| **Área de Productividad** | <ul><li>Como estudiante, quiero usar un temporizador Pomodoro (con soporte multi-pestaña) para gestionar mis tiempos.</li><li>Como estudiante, quiero gestionar mis tareas en un tablero Kanban para organizarme de forma ágil.</li><li>Como estudiante, quiero guardar enlaces útiles (Marcadores) directamente desde el área de estudio.</li></ul> |
| **Aprendizaje Activo** | <ul><li>Como estudiante, quiero crear mazos de Flashcards manualmente y generarlas usando Inteligencia Artificial.</li><li>Como estudiante, quiero repasar las Flashcards y registrar los resultados de mi aprendizaje.</li></ul> |
| **Finanzas y Admin** | <ul><li>Como estudiante, quiero ver el estado de mi cuota y recibir recordatorios de vencimiento.</li><li>Como estudiante, quiero acceder a un listado básico de beneficios y usar un formulario de contacto.</li><li>Como administrador, quiero ver la lista básica de alumnos registrados en el sistema.</li></ul> |

### 🟡 Release 2.0 (Mejoras Sociales, Usabilidad y Retención)
*(Objetivos a corto/mediano plazo para mejorar la experiencia)*

| Épica | Historias de Usuario (User Stories) |
| :--- | :--- |
| **Autenticación y Perfil** | <ul><li>Como usuario, quiero acceder a una versión Mobile 100% funcional y pulida para estudiar cómodamente desde mi celular.</li></ul> |
| **Gestión Académica** | <ul><li>Como estudiante, quiero compartir y sincronizar mis horarios con otros alumnos de la carrera.</li><li>Como estudiante, quiero ver un Ranking y Medallas (Gamificación) en mi panel de progreso para mantenerme motivado.</li></ul> |
